# OOP
Let's learn what object-oriented programming is.✏

## 소개 (특징)

- OOP는 객체의 관점에서 프로그래밍 하는 것을 의미합니다. 
- OOP는 객체들의 유기적인 관계를 통해 프로세스가 진행됩니다. 
- OOP는 4가지 특징이 있습니다. -> 추상화, 캡슐화, 상속, 다형성
- 이 4가지 특징을 이용하여 코드 재사용성을 증가시키고 유지보수를 감소시키는 장점을 얻습니다.
